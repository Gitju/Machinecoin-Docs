Get started with compiling the Machinecoin-Wallet and the Machinecoin-Daemon on GNU/Linux Debian 7.5:


Dynamically linked Machinecoin-Wallet and the Machinecoin-Daemon


  1. Download and Install VirtualBox and the VirtualBox Extension Pack.
  2. Download and import a GNU/Linux Debian 7.5 VirtualBox Appliance.
  3. Start your new virtual machine, login and open up a terminal and enter the following commands: 
    root@desktop:~$ apt-get update && apt-get upgrade && apt-get install build-essential libboost-all-dev libcurl4-openssl-dev libdb5.1-dev libdb5.1++-dev git libqt4-dev libqt4-core libqt4-gui libminiupnpc-dev qrencode libqrencode-dev
root@desktop:~$ su user
user@desktop:~$ cd ~ && git clone https://github.com/Gitju/Machinecoin.git && cd Machinecoin/src/ && make -f makefile.unix USE_IPV6=1 USE_UPNP=1 USE_QRCODE=1 && strip machinecoind && cd .. && qmake machinecoin-qt.pro USE_IPV6=1 USE_UPNP=1 USE_QRCODE=1 && make -f Makefile
  4. Now start the new compiled "machinecoin-qt" with a click on your mouse and close it again after the very first start. Next enter the following command in a terminal: 
    $vi ~/.machinecoin/machinecoin.conf
    When the config-file opens up just insert the following lines into the config-file and hit the save button. Don't forget to change <rpcusername> and <rpcpassword>. 
    rpcuser=<rpcusername>
rpcpassword=<rpcpassword>
rpcport=40332
server=1
listen=1
daemon=1
gen=0
rpcallowip=127.0.0.1
  5. Now start Machinecoin by clicking on the file "machinecoin-qt" inside of ~/machinecoin/.

Statically linked Machinecoin-Daemon


  1. Download and Install VirtualBox and the VirtualBox Extension Pack.
  2. Download and import a GNU/Linux Debian 7.5 VirtualBox Appliance.
  3. Start your new virtual machine, login and open up a terminal and enter the following commands: 
    root@desktop:~$ apt-get update && apt-get upgrade && apt-get install build-essential libboost-all-dev libcurl4-openssl-dev libdb5.1-dev libdb5.1++-dev git libqt4-dev libqt4-core libqt4-gui libminiupnpc-dev qrencode libqrencode-dev
root@desktop:~$ su user
user@desktop:~$ cd ~ && git clone https://github.com/Gitju/Machinecoin.git && cd Machinecoin/src/ && make -j 1 -f makefile.unix machinecoind STATIC=1 LDFLAGS="-static-libgcc -static-libstdc++" && strip machinecoind
  4. Now start the new compiled "machinecoind" and close it again after the very first start. Next enter the following command in a terminal: 
    $vi ~/.machinecoin/machinecoin.conf
    When the config-file opens up just insert the following lines into the config-file and hit the save button. Don't forget to change <rpcusername> and <rpcpassword>. 
    rpcuser=<rpcusername>
rpcpassword=<rpcpassword>
rpcport=40332
server=1
listen=1
daemon=1
gen=0
rpcallowip=127.0.0.1
  5. Now start the new compiled "machinecoind" again inside of ~/machinecoin/src/.
