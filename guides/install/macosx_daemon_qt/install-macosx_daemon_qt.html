1. Unzip "machinecoin-qt-mac.zip" to for example "/Users/<Username>/Downloads/"
2. Start "Machinecoin-Qt.app" with a mouse click and close it again after the very first start.
3. Copy "machinecoin.conf" to "/Users/<Username>/Library/Application\ Support/Machinecoin/machinecoin.conf" and change "<rpcusername>" and "<rpcpassword>" inside of it.
4. Start "Machinecoin-Qt.app" with a mouse click.
