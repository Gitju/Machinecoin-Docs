In "src/net.cpp" in the function "GetMyExternalIP(...)" there are several remote services defined to get the external ip.

To add such a service one must:

  1. Create a .php file on a webserver with support for php.
  2. Add the following content to this file: 
    <?php echo $_SERVER["REMOTE_ADDR"]; ?>
  3. Save that file and add the new remote service to "src/net.cpp"
